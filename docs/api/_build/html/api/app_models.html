Module contents¶

这是一个专为模型创建的app 所有的模型都放在这个app里 所有的模型都以models_开头，后面再加上具体内容或用途，例如： models_customuser.py, 为自定义用户而创建的模型 models_extra_user.py, 为用户模型增加一些extra的信息

然后在models.py文件里，import 需要的模型类
